    Local AI Agent

    A local AI agent powered by Retrieval Augmented Generation (RAG) for domain-specific question answering. It used langchain-ollama as the language model and langchain-chroma for vector storage.     Featured Project

    Sketch2Image Translation using GANs

    Developed an OpenCV-based method to convert original images into corresponding sketches. This dataset then served as the training ground for pix2pix, a conditional Generative Adversarial Network (cGAN), specifically designed for image-to-image translation. The architectural choices for the cGAN included a U-Net for the generator, enabling robust feature learning and reconstruction with skip connections, and a PatchGAN for the discriminator, which provided localized feedback to effectively guide the generator in producing high-fidelity translations from one image domain to another.

    • Python
    • OpenCV
    • Tensorflow
    • Keras
